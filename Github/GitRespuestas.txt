¿Quién inventó el sistema de control de versión Git y por qué?
El sistema de control de versión Git la desarrolló la comunidad de desarrollo de Linux (y en particular a Linus Torvalds, el creador de Linux). En ese entonces se desarrolló con el propósito de mejorar el desarrollo de software que en ese entonces se hacía por medio de parches y archivos.
